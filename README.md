# Dasa WordArt Poster Tool

90年代WordArt風の素材とダサい装飾でポスターを作る、ブラウザだけで動く静的ツールです。

## 使い方

`index.html` をブラウザで開きます。

- 背景色と模様を選択
- WordArt風テキストやダサ素材を追加
- 右パネルで文字、塗り色、文字色、フォント、回転、大きさを編集
- PNG保存でポスターを書き出し

## GitHub Pages

`.github/workflows/pages.yml` を同梱しています。

リポジトリの Settings → Pages で Source を `GitHub Actions` にすると、`main` ブランチへのpushで自動公開できます。
